# Promote Chat Answer

Use this workflow when a conversation produces a reusable answer, decision, or
synthesis.

## Steps

1. Decide whether the answer belongs in `wiki/questions/`, `wiki/syntheses/`, or
   an existing page.
2. Convert the answer into durable markdown with evidence links.
3. Mark conversation-derived claims explicitly when they are not source-backed.
4. Update related pages with backlinks.
5. Update [[index]].
6. Append an entry to [[log]].

## Rule Of Thumb

Promote answers that are likely to be searched for again, change how future work
should happen, or connect multiple existing pages.
