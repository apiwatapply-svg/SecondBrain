# Query Wiki

Use this workflow when answering a question from the vault.

## Steps

1. Read [[index]].
2. Search likely folders under `wiki/`.
3. Read relevant wiki pages.
4. Read raw sources only when wiki pages are thin, contested, or insufficient.
5. Answer with links to the pages used.
6. If the answer is reusable, create or update a durable page using
   [[templates/query-answer]] or [[templates/synthesis-page]].
7. Update [[index]] and [[log]] if durable pages changed.

## Answer Rules

- Separate sourced facts from inference.
- Cite pages with wikilinks.
- Say when the vault lacks enough evidence.
- Suggest sources to add when the question exposes a gap.
