# Compiled Knowledge

Compiled knowledge is information that has been processed once into a durable
structure so future questions can build from that structure instead of starting
from raw retrieval every time.

## Why It Matters

The wiki accumulates summaries, cross-links, contradictions, and syntheses. Each
new source should improve the existing structure.
