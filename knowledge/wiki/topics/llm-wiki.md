# LLM Wiki

An LLM Wiki is a persistent, interlinked markdown knowledge base maintained by
an LLM from curated sources and user questions.

## Core Pattern

- Raw sources are preserved separately.
- Wiki pages are compiled and maintained.
- The schema tells agents how to operate.
- Indexing and logging make the system navigable without heavy infrastructure.
