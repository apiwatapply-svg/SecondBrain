# Raw Sources

This directory is the evidence layer.

## Rules

- Treat accepted files in `raw/sources/` as immutable.
- Put new captures in `raw/inbox/` before ingest.
- Store downloaded images, PDFs, and other attachments in `raw/assets/`.
- Do not summarize here. Summaries belong in `wiki/sources/`.

## Folders

- [[raw/inbox/README]] - unprocessed captures.
- [[raw/sources/README]] - accepted source files.
- [[raw/assets/README]] - local attachments and media.
